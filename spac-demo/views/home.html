SPAC Mobile App Editor

This is the editor for the schedule content within the SPAC mobile application.

What you need to know

You can edit the following information by clicking the links below, or the tabs at the top of this page.

  • Group
  • Small Group
  • Pilates
  • Events

On these pages, you can add additional classes/events or remove classes/events from each day by clicking the 'Add Class' button or 'Remove' button. When you are done making your edits, please save the content by clicking the 'Save' button. This will update the files that the mobile app is linked to, and in-turn, the mobile application should reflect those changes instantly.
