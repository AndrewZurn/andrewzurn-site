== Description ==
A new and modern WordPress blog theme. Articles, pages, comments, menus and much more are part of this theme. Crafted with love by Monkey Themes.

* Mobile-first, Responsive Layout
* Custom Colors
* Custom Header
* Social Links
* Menu Description
* Post Formats
* The GPL v2 or later license. :) Use it to make something cool.

For more information about Monkey Themes please go to https://monkey-themes.com/

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
